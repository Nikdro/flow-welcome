Getting started

Here's how to get into the flow:

  1. Create your first package

    Go to {flow3CommandLinePath}
    and run ./flow3 package create MyPackage to create a package

  2. Create a controller

    Run ./flow3 kickstart controller MyPackage to generate a standard controller

  3. Remove the Welcome package

    Delete the Welcome package with ./flow3 package delete Welcome

  4. Test your controller

    Run {baseURI}MyPackage in your webbrowser

  5. Read the tutorial

    Switch over to the tutorial to get the most out of FLOW3.
