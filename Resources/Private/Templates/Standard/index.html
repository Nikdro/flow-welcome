Getting started

Here's how to get into the Flow:

  1. Create your first package

    Run flow3 package create MyPackage to create your first package

  2. Create a route and remove the Welcome package

    Routes are set up in Configuration/Routes.yaml

    Delete the Welcome package with flow3 package delete Welcome

  3. Create a controller

    Run flow3 generate controller MyPackage to generate a default controller

  4. Read the tutorial

    Switch over to the tutorial to get the most out of FLOW3.
